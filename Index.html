Act Fast Project

This App is called ACT fast and designed to educate people about emergencies that may happen. We picked 4 Emergencies: Flooding, Earthquake,Fire,Intruder. Users will first face the introduction screen and then the Option screen where they can pick what disaster they want to learn about, depending on what they pick-they will each be taken to a different screen where there are key details and important facts about what to do in a situation of an emergency. Once they are done reading up on it, they will click on a button that takes them to a quiz where they can test their knowledge!This app was developed using MIT App Inventor, using a block-based coding system to manage the app's logic. To make the home screen dynamic, I integrated an API that fetches and displays live weather data, like temperature and humidity based on the user's location. I used Layout components to organize the different emergency categories, such as Fire, Earthquake, Flood, and Lockdown, ensuring each had its own dedicated information page. For the interactive quizzes, I programmed Conditional Logic (If/Then blocks) to check if the user selected the correct checkboxes; this allows the app to trigger "Success" messages when answers are right or "Error" alerts if a question is missed or answered incorrectly. Finally, I used Notifier components to give users instant feedback, making the learning process feel like a real.
